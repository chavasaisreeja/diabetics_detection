What is a Decision Tree?

A Decision Tree is a Supervised Learning algorithm used for both Classification and Regression tasks. It is a tree-like structure where:

  • Internal nodes represent dataset features.
  • Branches define decision rules.
  • Leaf nodes represent the final classification or output.

How Does a Decision Tree Work?

The Decision Tree works by splitting the dataset into smaller subsets based on feature values. The process follows these steps:

  1. Select the best feature to split the dataset (using criteria like Gini Impurity or Entropy).
  2. Create branches based on feature values.
  3. Repeat the process for each subset until reaching a stopping condition (like minimum samples per leaf).
  4. Assign final outcomes to the leaf nodes.
